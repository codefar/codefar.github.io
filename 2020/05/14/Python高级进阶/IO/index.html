序列化

序列化: 把内存中数据序列化后的内容写入磁盘，或者通过网络传输到别的机器上。

反序列化：把变量内容从序列化的对象重新读到内存里称之为反序列化，即unpickling。

Python提供了pickle模块来实现序列化。
